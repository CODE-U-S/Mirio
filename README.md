# Mirio
2023 JAVA 프로젝트 'Mirio' 학교의 일상

## 🖥️ 프로젝트 소개
학교를 다니며 공부하고 시험보는 행위를 학생들의 시점에서 표현한 미니게임

<br/>

## 🕰️ 개발 기간
* 23.07.26일 - 23.11. 20일

### ⚙️ 개발 환경
- `Java 17`
- `JDK 17.0.8`
- **IDE** : Eclipse
